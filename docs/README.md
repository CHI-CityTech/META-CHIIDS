# CHIIDS Documentation

This directory is the primary documentation hub for META-CHIIDS.
It brings together platform context, architecture references, integration guides, and planning documents for the CHIIDS project database backend.

Most sections in this docs hub are conceptual: they define vocabulary, architecture, governance context, and integration pathways.
The `projectdb/` section is different by design: it is the implementation-oriented track where backend scope, deliverables, and execution resources are documented for building the operational CHI project database.

Use this index to navigate core CHIIDS references:

- **Overview:** [overview.md](overview.md) — one-page abstract of CHIIDS
- **Usage Guide:** [usage.md](usage.md) — how to search, filter, and export from Datasette
- **Architecture:** [architecture/](architecture/) — system design, registry, and structure
- **Integrations:** [integrations/](integrations/) — Datasette, OpenLab, WorldAnvil guides
- **Reference:** [reference/](reference/) — glossary, ecosystem, data model, integration paths, workflows
- **Project Database:** [projectdb/](projectdb/) — initial proposal, briefing, researcher metadata, and working resources
