# CHIIDS Meta-Project Glossary

Define terms specific to this Layer 1 meta-project. For Layer 0 terminology, see [docs/reference/glossary.md](../docs/reference/glossary.md).

- Meta-Project: A project whose outputs define or orchestrate other projects.
- Registry: The catalog of projects and dependencies managed by CHIIDS.
- Dependency Graph: The representation of project-to-project dependencies.
